📖 How to Use (No Separate Page Needed)

Add this header to every API request:

Authorization: Bearer YOUR_API_KEY

POST — Create a Post (for automation)

POST https://yourdomain.com/api/v1/posts
{
  "title": "SRK Ki Nayi Film!",
  "content": "Full article here...",
  "category": "बॉलीवुड",
  "image": "https://img.jpg",
  "status": "published"
}

Python Example

import requests
API_KEY = "fb_live_your_key"
r = requests.post("https://yourdomain.com/api/v1/posts",
    headers={"Authorization": f"Bearer {API_KEY}"},
    json={"title":"My Post","content":"...","category":"बॉलीवुड"})
print(r.json())
